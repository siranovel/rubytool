rubytool
========
c言語からrubyのライブラリを呼び出すツール集です

## Description ##
### ツールの種類 ###
* rbmath  
  rubyのMath情報を表示するプログラムです  
  表示項目
  - PI
  - E
  - 平方根  
    ２、３、５、７
